# Ling Ding Portfolio Website

An academic portfolio website for Ling Ding, SSHRC Doctoral Fellow and Ph.D.
student at the University of Toronto.

## What is included

- Responsive homepage
- Research, education, publications, and work sections
- Skills and methods section
- Contact section
- Portrait image in `public/ling-ding-portrait.png`

## Local development

```bash
npm install
npm run dev
```

## Build check

```bash
npm run build
```

## Customize next

- Add more publications and project links in `app/page.tsx`
- Add LinkedIn, Google Scholar, or ORCID links when ready
- Replace or update the portrait in `public/ling-ding-portrait.png`
